# 🚀 About Me

Backend developer focused on designing scalable, maintainable, and production-ready applications using **ASP.NET Core**, **C#**, **Entity Framework Core**, and **SQL Server**.

My engineering approach is centered on building software that remains clean, extensible, and easy to evolve as requirements grow. I apply **Clean Architecture**, **Onion Architecture**, SOLID principles, and layered design not as templates, but as practical tools for managing complexity and improving long-term maintainability.

I have hands-on experience developing secure RESTful APIs, implementing authentication and authorization, integrating Entity Framework Core, and designing modular backend solutions with a strong emphasis on code quality, consistency, and architectural integrity.

Currently working as a freelance backend developer on **Horis Geneve**, a multi-tier partnership and investment platform featuring role-based authorization, hierarchical user management, commission processing, and automated payout workflows.
